JSON学习

1 JSON入门

1.1 介绍

  • JSON（JavaScript Object Notation）：JS对象符号，是一种轻量级的书交换格式（相对比于xml）。
  • 数据交换一般为客户端和服务端之间进行数据交换

1.2 数据格式

  • 特点：
    • json是一个对象，用{}来包裹数据
    • json的数据，存储形式类似key-value形式
    • json的key值要用""括起来
    • json的value值支持数字、字符串、布尔型、json对象
    • json数据之间用,进行隔开
